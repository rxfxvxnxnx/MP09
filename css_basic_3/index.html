En aquest exercici es tracta de crear uns estils nous de classe, que incloguin una imatge de fons, de manera que es puguin aplicar posteriorment als enllaços de la pàgina, per diferenciar els vincles a arxius de tipus pdf, de tipus word, de tipus document de text, de tipus música i de tipus imatge. És a dir, cada vegada que aparegui en el lloc web un enllaç a algun dels tipus d'arxius anteriors, en aplicar la classe apropiada, apareixerà també una imatge que indicarà a l'usuari el tipus de vincle. L'objectiu és definir 5 estils CSS (de classe) que aplicats als enllaços quedin de la següent manera:

  • Documento PDF
  • Documento Word
  • Documento de texto
  • Musica
  • Imagen
